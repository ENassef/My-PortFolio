# 🚀 Portfolio | Nassef

> **Crafting Digital Experiences with Code & Creativity**

A modern, responsive portfolio website built with cutting-edge web technologies, showcasing my journey as a full-stack developer and my passion for creating exceptional digital solutions.

## ✨ Features

### 🎨 **Design & UX**
- **VSCode IDE Theme** - A unique, developer-inspired dark theme design
- **Responsive Design** - Seamless experience across all devices (Mobile-first approach)
- **Smooth Animations** - Powered by Framer Motion, GSAP, and AOS
- **Interactive Elements** - Hover effects, transitions, and micro-interactions

### 🛠️ **Technical Stack**
- **Frontend**: React 19.1.0 with Vite 7.0.4 for lightning-fast development
- **Styling**: TailwindCSS 4.1.11 for utility-first CSS
- **Routing**: React Router DOM 7.7.1 for seamless navigation
- **Forms**: Formik with Yup validation
- **Icons**: FontAwesome, Lucide React, React Icons
- **SEO**: React Helmet for optimal search engine optimization
- **Analytics**: Vercel Analytics integration

### 📱 **Pages & Sections**
- **Home** - Hero section with animated introduction
- **Projects** - Showcase of real-world projects and assignments
- **About** - Professional background and skills
- **Contact** - Functional contact form with validation
- **Theme** - Custom theme switcher for personalized experience

## 🛠️ Installation & Setup

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Clone & Run

```bash
# Clone the repository
git clone https://github.com/ENassef/My-PortFolio.git

# Navigate to project directory
cd My-PortFolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
